# Spring Boot AWS S3 Image Upload

image upload application built using spring-boot and aws s3 bucket and angularjs.

## config

replace the following credentials in the `application.properties`

```
client_key
secret_key
bucket_name
region
```

## run

use the following command to run the application

`mvn spring-boot:run`

and access `http://localhost:8080` to view the application.
